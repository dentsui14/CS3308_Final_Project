Welcome#

Welcome to Intro to Computing and AI for Business Professionals! This course is designed for business professionals who regularly use computers but want to better understand the fundamentals of computing and AI to be able to make more informed technological solutions. Other than basic use of a computer, there are no prerequisites for this course.

While it is strongly recommended to complete the computing module before the AI module, they are technically independent of each other.

Important: How to run live code#

Live code underpins many of the lessons in this course. While it may not be needed for each lesson it is strongly recommended that you start it at the start of every lesson. You will have to restart it for every new lesson, though the first time will take the longest to start.

Follow along in this lesson.

  1. You will see a rocket symbol in the top right. Click, don’t hover, on it.

  1. On the dropdown menu, click live code.

  1. You will see the following box appear at the top of the lesson. This means that the Python interpreter that will be used to run all code is booting up.

  1. When the box changes from “launching” to “ready”, you are good to go!

  1. If everything is correct you should see a “run” button under the code block below.

  2. Press run to display the message.

print("You are now ready to begin the course. Good luck and happy coding!")
